AI 助手自动代理功能

本网站的 AI 助手功能可以在 GitHub Pages 环境中自动连接到您本地运行的 Ollama 服务，无需任何额外配置。

工作原理

当您通过 GitHub Pages 访问本网站时，AI 助手会：

  1. 自动检测您正在使用 HTTPS 环境
  2. 尝试多个公共 CORS 代理服务
  3. 选择一个可用的代理服务连接到您本地的 Ollama

使用要求

要使用此功能，您只需要：

  1. 在本地安装并运行 Ollama 服务
  2. 确保 Ollama 服务在默认端口 (11434) 上运行
  3. 确保您的浏览器允许跨域请求

故障排除

如果连接失败：

  1. 确认 Ollama 服务正在本地运行
    • 在浏览器中访问 http://localhost:11434/api/tags 应该显示可用模型列表
  2. 点击 AI 助手中的”重新尝试连接”按钮
  3. 如果仍然失败，可能是所有公共代理服务都不可用，请稍后再试

隐私说明

使用公共代理服务时，您的请求和 Ollama 的响应将通过该服务传输。虽然内容仍然在您的设备和本地 Ollama 之间传输，但代理服务理论上可以看到这些数据。

如果您对隐私有严格要求，建议在本地环境中使用本网站。
